Sql deployer

Sql Deployer e' un plugin concepito per agganciare al normale processo di build di un progetto maven l'esecuzione di codice sql. Il codice SQL eseguito viene memorizzato su una base dati, che normalmente coincide con la stessa che e' stata aggiornata.

Requisiti di sistema

Il plugin e' stato compilato con java 1.5, ergo e' richiesta almeno una virtual machine 1.5.
